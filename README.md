#  Global Happiness Report Analysis

A data analysis and visualization project using the **WHR_2015** dataset. This project explores the factors influencing happiness across different countries using Python, Pandas, Matplotlib, and Seaborn.

---

##  Project Overview

The objective of this project is to analyze the World Happiness Report dataset and identify patterns between happiness score and various socio-economic factors such as GDP per capita and social support.

## Objectives

- Perform data cleaning and exploration.
- Analyze happiness scores across countries.
- Identify the happiest countries.
- Study the relationship between GDP per capita and happiness.
- Analyze social support versus happiness.
- Create meaningful visualizations for better insights.

- Iceland ranks among the happiest countries.
